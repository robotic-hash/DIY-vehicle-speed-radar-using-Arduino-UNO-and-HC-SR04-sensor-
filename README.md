
# Arduino Speed Radar 🚗

This project measures the speed of an object using two HC-SR04 sensors and displays the result on an LCD.

## 🔧 Components
- Arduino UNO
- 2x HC-SR04
- LCD I2C
- Buzzer

## ⚙️ How it works
The system calculates the time between two detections and computes the speed.
